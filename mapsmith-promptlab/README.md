# MapSmith

## Project Overview
MapSmith is a prompt-engineering project designed to build a prompt-driven system for AI-assisted D&D battle map generation and evaluation. This project explores how carefully crafted prompts can guide AI models to produce coherent, detailed, and usable battle maps for tabletop gaming.

This repository is currently in the **scaffold and setup phase**. Core project requirements and detailed specifications will be added by referencing a project guidelines document.

## Project Structure
- `data/` — Test cases, prompt templates, and evaluation data
- `docs/` — Project proposal, final report, presentation, and references
- `src/` — Core Python modules for the project
- `notebooks/` — Jupyter notebooks for exploration and analysis
- `results/` — Generated images and exports
- `tests/` — Unit tests and test utilities

For an overview of folder purposes, see [PROJECT_NOTES.md](PROJECT_NOTES.md).
